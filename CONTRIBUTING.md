# Contribuer au projet

Merci de l'intérêt pour ce projet propulsé par **JARVIS Turbo** (Turbo31150).

## Processus

1. Fork + branche : `git checkout -b feat/ma-feature`
2. Effectue tes modifications
3. Signe tes commits (DCO) : `Signed-off-by: Nom <email>`
4. Pull Request vers `main`

## DCO — Developer Certificate of Origin

En soumettant une contribution, tu certifies que c'est sous licence **MIT** et que tu en es l'auteur.

## Style de code

- Python : PEP 8 + Black | TypeScript/JS : ESLint + Prettier
- Commits : `feat:` `fix:` `docs:` `chore:`
